# Circle Programmable Wallets

In this campaign, you'll explore Programmable Wallets with Circle, gaining expertise in managing secure Web3 wallets across various blockchains like Ethereum, Avalanche, and Polygon using developer-friendly REST APIs, along with the opportunity to build a custom wallet using the Programmable Wallet SDK, unlocking NFT badges upon completion.
